# MySpace - 个人博客 (Web 大作业)

## 项目简介

这是一个基于原生 HTML、CSS 和 JavaScript 构建的个人博客网站。该项目作为 Web 课程期末大作业，展示了基础的网页布局、交互特效以及 3D Canvas 动画等技术。

网站包含个人介绍、文章分享、游戏开黑好友位展示，以及一个特别制作的 **圣诞特辑 (3D 光子圣诞树)** 页面。

## 最初版功能

### 共有功能

- **导航栏**: 包含下拉菜单，支持快速跳转各子页面。

- **实时时钟**: 页脚包含一个动态运行的实时时间显示。

### 1. 首页 (`index.html`)

- **轮播图**: 首页顶部展示动态轮播图。

- **最新动态**: 展示网站的最新更新和公告卡片。

### 2. 圣诞特辑 - 3D 光子圣诞树 (`gallery.html`)

这是一个基于 HTML5 Canvas 实现的高级视觉特效页面：

- **3D 粒子系统**: 由圣诞树、烟花和飘雪组成的 3D 圣诞树。

- **交互体验**:

- **旋转**: 拖动鼠标可以 360 度旋转观察圣诞树。

- **烟花特效**: 背景伴有随机升起和绽放的烟花。

### 3. 我的文章 (`blog.html`)

- **文章列表**: 聚合展示了站长撰写的技术教程与生活分享。

- **精选内容**:

- **AI 福利**: 教授如何获取 Google Gemini Advanced 权益。

- **效率工具**: Obsidian + GitHub + AI 构建永久知识库的方案。

- **数码折腾**: 小米平板 5 刷 Windows 11 双系统的保姆级指南。

- **侧边栏**: 提供文章分类目录（羊毛福利、效率工具、技术教程）。

### 4. 个人简介 (`about.html`)

- 展示个人头像与简介。

- **技能条**:动态展示 HTML, C/C++, Java, Python 等技能掌握情况。

- **GitHub 链接**: 快速访问开发者的 GitHub 主页。

### 5. 联系我 (`contact.html`)

- **留言板**: 提供一个模拟的在线留言表单（包含姓名、邮箱、主题、内容验证）。

- **社交名片**: 展示站长的多渠道联系方式，包括微信 (WeChat)、抖音 (Douyin) 和 QQ 的二维码名片。

### 6. 找我开黑 (`game.html`)

- 展示站长常玩的游戏（守望先锋、炉石传说、我的世界）。

- 提供游戏 ID 和相关信息，方便访客添加好友。

### 7. 小巧思

- **全局挂件**: 页面右下角包含一个不停旋转的尼禄挂件，鼠标悬停会加速旋转。

- **全局雪花**: 全站（除警告页外）覆盖有飘落的雪花特效，营造冬日氛围。

- **圣诞帽**: 个人简介页面的头像上方有一个圣诞帽 ，增加圣诞元素。

## 更新内容

### 1. 高危区域（新增） (`danger.html`)

- 一个趣味性的警告页面。

- 包含模态框 (Modal) 交互，经过多次确认后会跳转到特定视频链接。

### 2. 圣诞树效果 (`gallery.html`)

- **灯光效果（修改）**: 将灯光效果改为了盘旋的灯带效果。

- **灯光切换（新增）**: 点击鼠标左键/右键可切换灯带颜色（红、黄、蓝、紫、粉）。

- **星河模式（新增）**: 滚动鼠标滚轮向上，圣诞树粒子会炸裂散开形成“星河”；向下滚动则聚拢回圣诞树。

- **背景音乐（新增）**: 自动播放 Jingle Bells 圣诞音乐。

### 3. 首页 (`index.html`)

- **文案（修改）**:圣诞节已过，对应的圣诞文案已进行修改；新增更新提醒文案。
